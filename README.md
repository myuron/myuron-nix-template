# Overview
- flake.nix template for project

# Usage
## download command
```bash
nix flake init --template "github:myuron/myuron-nix-template#<template>"
```

## templates
|template|description|
|---|---|
|go|includes Go|
|go-with-agentskills|includes Go and Agent-Skills|
|typescript|includes typescript|
|typescript-with-agentskills|includes typescript and Agent-Skills|
